# Kalkulator c++
Prosty kalkulator, który umożliwi wykonywanie obliczeń takich jak: dodawanie, odejmowanie, 
mnożenie, dzielenie, obliczenie pierwiastka kwadratowego, obliczenie x do potęgi y. Program pyta o rodzaj działania (+, -, …) i wartości na których należy wykonać operacje.  
Program powinien działać w pętli do momentu, kiedy użytkownik  nie podejmie decyzji o zakończeniu programu.
